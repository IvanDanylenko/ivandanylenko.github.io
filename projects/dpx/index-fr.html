Qu'est-ce que DLX?

DiplexCoin (DLX) est une monnaie électronique open source alimentée par l'algorithme X13.


DLX peut être utilisé pour des paiements instantanés partout dans le monde.

Découvrez la technologie X13!

La pièce peut être facilement intégrée dans n'importe quel projet en tant que paiement via notre API

Qu'est-ce que la preuve d'enjeu (PoS)?

C'est l'un des types de minage (PoS - preuve d'enjeu), qui utilise un algorithme de distribution de pièces dans le réseau. Dans le processus de minage (pourchasser les gens), vous obtenez une récompense sous forme de pièces supplémentaires. L'idée derrière la preuve de participation est de résoudre le problème de la preuve de travail associé à des coûts énergétiques élevés et des investissements financiers.
